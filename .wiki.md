# Project Summary
The project is an advanced freight management system that optimizes logistics and shipping operations across various transportation modes, including sea, rail, and truck freight. It provides businesses with essential tools for cost calculation, user account management, quotation generation, and historical data tracking. By enhancing efficiency and transparency in the freight industry, this system enables businesses to reduce costs and streamline workflows through an intuitive user interface, ultimately improving their operational effectiveness.

# Project Module Description
The project consists of several functional modules:
- **User Management**: Manages user authentication and role assignments.
- **Freight Calculation**: Computes freight costs and maintains historical data.
- **Audit Logging**: Tracks changes to freight data for traceability.
- **Data Management**: Manages freight-related data entities, including DTHC, DPCost, and weight surcharge rules.
- **Port and Destination Tables**: Manages freight rates for rail and truck transport across various destinations.
- **Combined Freight Table**: Handles combined freight rates and operations.
- **Freight Calculator**: Provides a user interface for calculating freight estimates.
- **Cost Input Form**: Allows users to input additional costs for freight calculations.
- **Cost Result Table**: Displays results of freight calculations and cost breakdowns.
- **User Dashboard**: Displays user statistics and recent calculations.
- **Agent Management**: Manages rail and truck agents with a focus on usability.

# File Description Inventory
- **AuditLogTable.tsx**: Displays audit logs related to freight operations.
- **BorderDestinationTable.tsx**: Manages truck freight rates with CRUD functionalities.
- **CombinedFreightTable.tsx**: Handles combined freight rates and operations, sorted by creation date.
- **DPCostTable.tsx**: Manages direct port cost entries, sorted by creation date.
- **DTHCTable.tsx**: Handles D/O (DTHC) data for terminal handling charges, sorted by creation date.
- **PortBorderTable.tsx**: Manages rail freight rates, sorted by creation date.
- **SeaFreightTable.tsx**: Manages sea freight data, sorted by creation date.
- **WeightSurchargeTable.tsx**: Manages weight surcharge rules, sorted by creation date.
- **DestinationTable.tsx**: Handles destination freight management.
- **PortTable.tsx**: Manages port information.
- **ShippingLineTable.tsx**: Manages shipping line data.
- **CostInputForm.tsx**: Provides a UI for inputting costs associated with freight calculations.
- **CostResultTable.tsx**: Displays results of freight calculations and cost breakdowns.
- **CostCalculatorWithTabs.tsx**: Implements the freight cost calculator with tab functionality.
- **CostCalculatorHeader.tsx**: Updated header for the cost calculator, matching the compact style of the dashboard.
- **QuotationDialog.tsx**: Updated dialog for quotations, designed to be more compact and user-friendly.
- **Layout.tsx**: Defines the layout structure of the application.
- **FreightContext.tsx**: Contains the context for managing freight-related data.
- **freightCalculations.ts**: Implements the cost calculation logic with validation for valid freight rates.
- **UserDashboard.tsx**: Displays user statistics and recent calculations.
- **UserManagement.tsx**: Manages user accounts and roles.

# Usage
1. **Install Dependencies**: Run the command to install the necessary dependencies.
2. **Build the Project**: Use the build command to compile the project.
3. **Run the Application**: Start the application using the appropriate command.
4. **Interact with the Calculator**: Utilize the UI to input POL, POD, weight, and additional costs to calculate freight estimates, including the option to select historical dates.
