# Project Summary
This project is an advanced freight cost calculator tailored for logistics companies, enabling them to optimize shipping cost estimations. By factoring in variables such as ports of loading (POL), ports of discharge (POD), weight, and additional fees, the application enhances operational efficiency and decision-making. Users can compute and compare freight costs across various shipping methods. Recent updates have improved user experience, particularly in the CostResultTable component, which now emphasizes the most economical options with enhanced visual indicators.

# Project Module Description
The project is composed of several functional modules:
- **CostCalculatorWithTabs**: Core component for calculating and displaying freight costs with a tabbed interface.
- **AdminDashboard**: Interface for managing freight-related data, including agents and ports.
- **AgentSeaFreightTable**: Displays statistics of sea freight agents.
- **ShippingLineTable**: Manages and displays shipping line information.
- **DTHCTable**: Manages Destination Terminal Handling Charge (D/O) entries.
- **CombinedFreightTable**: Manages combined freight rates.
- **BorderDestinationTable**: Manages freight rates for border destinations.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **DPCostTable**: Manages direct port cost information.
- **CostCalculatorHeader**: Header component for the cost calculator.
- **CostResultTable**: Displays optimized cost calculation results, now highlighting agent codes and the lowest prices with a "BEST" label.
- **CalculationHistory**: Manages and displays historical calculation data.
- **SeaFreightDialog**: Dialog for selecting sea freight options.
- **CostInputForm**: Updated form for inputting cost-related data in a card layout.
- **RailAgentTable**: Displays rail freight agents, including agent codes.
- **TruckAgentTable**: Displays truck freight agents, including agent codes.
- **LoginForm**: User authentication form with enhanced user experience.
- **TimeMachineDialog**: Enhanced dialog for selecting historical dates for freight rate changes.
- **PortBorderTable**: Manages freight rates for border crossings.
- **QuotationDialog**: New dialog for generating quotations based on calculated costs and exporting to Excel.
