# Project Summary
Shadcn-UI is an advanced logistics and shipping web application designed to optimize freight management for businesses. The platform enhances operational efficiency with features such as cost estimation, historical data analysis, and user role management. It integrates user authentication through Supabase, offers a sophisticated Cost Calculator, and provides real-time freight options. Recent updates focus on improving user experience, particularly in the Admin Dashboard with enhanced styling and functionality.

# Project Module Description
- **AdminDashboard**: Centralized management of freight data and statistics with enhanced user interface elements.
- **UserDashboard**: Access to freight calculations and historical data.
- **CombinedFreightTable**: Management of combined freight rates with improved styling.
- **SeaFreightTable**: Handles sea freight rates with updated terminology and features.
- **AgentSeaFreightTable**: Manages agent-specific sea freight rates with improved dialogs.
- **FreightTable**: Overview of freight routes and costs with search and pagination features.
- **AddRouteForm**: Interface for adding new freight routes.
- **CostCalculator**: Tool for calculating shipping costs with advanced options.
- **WeightSurchargeTable**: Management of weight surcharge rules.
- **OtherCostsTable**: Handling of additional freight transport costs.
- **LoginForm**: User authentication interface.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Inline editing functionality for table data.
- **UserManagement**: Manage user roles and permissions, with updates for improved functionality.
- **QuotationDialog**: Generate and export quotations.
- **AuditLogTable**: Displays detailed audit logs.
- **TimeMachineDialog**: View historical audit logs.
- **ValidityPeriodInput**: Manage freight rate validity periods.
- **SupabaseIntegration**: Handles connection to Supabase.
- **SupabaseMigration**: Manages initial data migration to Supabase.
- **CheckDuplicateRailFreight**: Identifies duplicate rail freight entries.
- **ExcelExport**: Export quotations to Excel with enhanced styling.
- **DestinationTable**: Manage destination entries with improved functionality.
- **CalculationHistory**: New component for managing calculation history.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for static typing.
- **Shadcn-UI**: Component library for UI elements.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web projects.
- **xlsx**: Library for Excel file manipulation.
- **xlsx-js-style**: Enhanced library for styled Excel file manipulation.
- **Supabase**: Backend as a service for database management and real-time data handling.
- **Vercel**: Deployment platform for hosting applications.
