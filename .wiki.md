# Project Summary
Shadcn-UI is a logistics and shipping web application designed to optimize freight management for businesses. It focuses on cost estimation, historical data analysis, and user role management. The application utilizes Supabase for backend services, enhancing user authentication and providing a sophisticated Cost Calculator. Recent updates have improved deployment settings for efficient Single Page Application (SPA) routing, aiming to deliver an enhanced user experience.

# Project Module Description
- **AdminDashboard**: Interface for managing freight data and statistics.
- **UserDashboard**: Interface for users to view freight calculations and history.
- **CombinedFreight**: Manages combined freight rates with versioning and search capabilities.
- **FreightTable**: Handles freight routes and costs, featuring search filters and pagination.
- **AddRouteForm**: Interface for adding new freight routes.
- **CostCalculator**: Tool for calculating shipping costs, recently enhanced.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **OtherCostsTable**: Handles additional freight transport costs.
- **AgentSeaFreightTable**: Manages sea freight rates for agents with updated filters.
- **BorderDestinationTable**: Manages freight rates for border destinations.
- **PortBorderTable**: Updates freight rates for specific ports.
- **RailAgentTable**: Manages rail agent information.
- **TruckAgentTable**: Manages truck agent details.
- **DTHCTable**: Manages D/O (DTHC) costs.
- **LoginForm**: User authentication interface, recently updated.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Allows inline editing for table data.
- **UserManagement**: Manages user roles and permissions.
- **QuotationDialog**: Generates and exports quotations.
- **AuthContext**: Manages user authentication state.

- **ShippingLineTable**: Manages shipping line data.
- **AuditLogTable**: Displays audit logs, updated to show 5 items per page.

- **ValidityPeriodInput**: Manages freight rate validity periods.
- **SeaFreightTable**: Manages sea freight rates with version tracking.
- **SupabaseIntegration**: Manages Supabase connection.

- **CheckDuplicateRailFreight**: Identifies duplicate rail freight entries.
