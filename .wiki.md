# Project Summary
Shadcn-UI is a logistics and shipping web application designed to streamline freight management for businesses. It provides functionalities such as cost estimation, historical data analysis, and user role management, leveraging Supabase for user authentication. The platform includes a Cost Calculator for real-time shipping options, a User Management system, and an Admin Dashboard. Recent enhancements have improved the display logic for freight rates, particularly showing "$0" for zero values, which increases calculation accuracy and user experience.

# Project Module Description
- **AdminDashboard**: Manages freight data and statistics.
- **UserDashboard**: Portal for freight calculations and historical data.
- **CombinedFreightTable**: Handles combined freight rates with POL dropdown.
- **SeaFreightTable**: Manages sea freight rates with updated terminology and display logic for zero values.
- **AgentSeaFreightTable**: Manages agent-specific sea freight rates, allowing rates of 0 and correct display of local charges.
- **FreightTable**: Overview of freight routes and costs with search and pagination.
- **AddRouteForm**: Interface for adding new freight routes.
- **CostCalculatorWithTabs**: Calculates shipping costs, now displaying "$0" for zero values.
- **DebugLLocal**: Component for debugging local values.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **OtherCostsTable**: Handles additional freight costs.
- **LoginForm**: User authentication interface, includes version information.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Inline editing functionality for table data.
- **UserManagement**: Manage user roles and permissions.
- **QuotationDialog**: Generate and export quotations.
- **AuditLogTable**: Displays detailed audit logs.
- **TimeMachineDialog**: View historical audit logs.
- **ValidityPeriodInput**: Manage freight rate validity periods.
- **SupabaseIntegration**: Connects to Supabase.
- **SupabaseMigration**: Manages initial data migration to Supabase.
- **CheckDuplicateRailFreight**: Identifies duplicate rail freight entries.
- **ExcelExport**: Export quotations to Excel.
- **DestinationTable**: Manage destination entries.
- **BorderDestinationTable**: Manages border destination freight rates.
- **DPCostTable**: Enhanced with Package icon.
- **FreightLoaders**: Loads various freight-related data.
- **FreightCalculations**: Module for calculating costs and managing freight values.
- **VersionInfo**: Manages versioning and update history displayed in the login interface.
