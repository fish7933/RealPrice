# Project Summary
Shadcn-UI is a logistics and shipping web application aimed at enhancing freight management for businesses. The platform offers tools for cost estimation, historical data analysis, and user role management, thereby improving operational efficiency. Key features include user authentication via Supabase, a sophisticated Cost Calculator, and a dynamic freight options viewer. Recent updates have refined D/O (DTHC) management, ensuring real-time updates of carrier data during record modifications to uphold data integrity.

# Project Module Description
- **AdminDashboard**: Manage freight data and statistics.
- **UserDashboard**: View freight calculations and history.
- **CombinedFreight**: Handle combined freight rates with versioning and search.

- **AddRouteForm**: Add new freight routes.
- **CostCalculator**: Calculate shipping costs with advanced features.
- **WeightSurchargeTable**: Manage weight surcharge rules.
- **OtherCostsTable**: Handle additional freight transport costs.
- **AgentSeaFreightTable**: Manage sea freight rates for agents with updated filters.
- **BorderDestinationTable**: Manage freight rates for border destinations.
- **PortBorderTable**: Update freight rates for specific ports.
- **RailAgentTable**: Manage rail agent information.
- **TruckAgentTable**: Manage truck agent details.
- **DTHCTable**: Manage D/O (DTHC) costs with improved layout for carrier selection and D/O amount input, now allowing carrier changes in version updates.
- **LoginForm**: User authentication interface.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Inline editing for table data.
- **UserManagement**: Manage user roles and permissions.
- **QuotationDialog**: Generate and export quotations.
- **AuthContext**: Manage user authentication state.
- **FreightContext**: Handle freight-related data, including version tracking and cost calculations.
- **ShippingLineTable**: Manage shipping line data.
- **AuditLogTable**: Display audit logs.
- **TimeMachineDialog**: View historical audit logs.
- **ValidityPeriodInput**: Manage freight rate validity periods.
- **SeaFreightTable**: Manage sea freight rates with version tracking.
- **SupabaseIntegration**: Manage Supabase connection.
- **SupabaseMigration**: Handle initial data migration to Supabase.
- **PortBorderFreightTable**: Manage port border freight rates.
- **CheckDuplicateRailFreight**: Identify duplicate rail freight entries.
- **FindDuplicateRailFreight**: Check for duplicate rail freight entries.
- **ListTables**: List tables in the Supabase database.
- **ValidityHelper**: Utility for managing validity periods.
- **ExcelExport**: Export quotations to Excel with enhanced styling.
- **ResetSuperadminPassword**: Edge Function for resetting the superadmin password.
- **VercelIntegration**: Updated deployment configurations to fix 404 errors.
- **CalculationHistory**: Manage calculation history.
- **ProfileSettings**: Manage user profile settings, including password changes.

- **Split FreightContext**: Modularized FreightContext into smaller, maintainable files.
