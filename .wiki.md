# Project Summary
Shadcn-UI is a logistics and shipping web application aimed at enhancing freight management for businesses. It provides features such as cost estimation, historical data analysis, and user role management, which collectively improve operational efficiency. The application leverages Supabase for user authentication and incorporates a dynamic Cost Calculator for real-time shipping options. Recent developments have focused on refining user experience in User Management and the Admin Dashboard, with notable improvements in styling and functionality.

# Project Module Description
- **AdminDashboard**: Interface for managing freight data and statistics.
- **UserDashboard**: Portal for freight calculations and historical data.
- **CombinedFreightTable**: Manages combined freight rates with improved styling.
- **SeaFreightTable**: Manages sea freight rates with updated terminology.
- **AgentSeaFreightTable**: Handles agent-specific sea freight rates.
- **FreightTable**: Overview of freight routes and costs with search and pagination.
- **AddRouteForm**: Interface for adding new freight routes.
- **CostCalculatorWithTabs**: Tool for calculating shipping costs with tabbed navigation, now enhanced to correctly display L.LOCAL values and indicate agent-specific sea freight with a star.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **OtherCostsTable**: Handles additional freight transport costs.
- **LoginForm**: User authentication interface.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Inline editing functionality for table data.
- **UserManagement**: Manage user roles and permissions.
- **QuotationDialog**: Generate and export quotations.
- **AuditLogTable**: Displays detailed audit logs.
- **TimeMachineDialog**: View historical audit logs.
- **ValidityPeriodInput**: Manage freight rate validity periods.
- **SupabaseIntegration**: Connects to Supabase.
- **SupabaseMigration**: Manages initial data migration to Supabase.
- **CheckDuplicateRailFreight**: Identifies duplicate rail freight entries.
- **ExcelExport**: Export quotations to Excel.
- **DestinationTable**: Manage destination entries.
- **BorderDestinationTable**: Manages border destination freight rates.
- **DPCostTable**: Updated to include Package icon for enhanced functionality.
- **FreightLoaders**: Loads various freight-related data.
- **FreightCalculations**: Calculates costs with tracking of expired rates.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for static typing.
- **Shadcn-UI**: Component library for UI elements.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web projects.
- **xlsx**: Library for Excel file manipulation.
- **xlsx-js-style**: Enhanced library for styled Excel file manipulation.
- **Supabase**: Backend as a service for database management and real-time data handling.
- **Vercel**: Deployment platform for hosting applications.
