# Project Summary
The project is an advanced freight cost calculator tailored for logistics companies, providing precise shipping cost estimates by factoring in parameters like ports of loading (POL), ports of discharge (POD), weight, and additional charges. This application enhances decision-making and operational efficiency in the logistics sector by allowing users to calculate and compare freight costs across various shipping methods. Recent updates have improved the user interface with vibrant gradients, animations, and a refined visual hierarchy, significantly enhancing the user experience.

# Project Module Description
The project consists of several functional modules:
- **CostCalculatorWithTabs**: Core component for calculating and displaying freight costs with a tabbed interface.
- **AdminDashboard**: Interface for managing freight-related data, including agents, ports, and costs.
- **AgentSeaFreightTable**: Displays statistics of sea freight agents.
- **DTHCTable**: Manages Destination Terminal Handling Charge (D/O) entries and their versions.
- **CombinedFreightTable**: Manages combined freight rates and their details.
- **BorderDestinationTable**: Manages freight rates for border destinations.
- **WeightSurchargeTable**: Manages rules for weight surcharges.
- **DPCostTable**: Manages direct port cost information.
- **CostCalculatorHeader**: New header component for the cost calculator.
