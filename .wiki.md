# Project Summary
Shadcn-UI is a logistics and shipping web application designed to optimize freight management for businesses. It improves operational efficiency through features like cost estimation, historical data analysis, and user role management. The platform utilizes Supabase for user authentication and includes a dynamic Cost Calculator that provides real-time shipping options. Recent updates have focused on enhancing user experience, especially in User Management and the Admin Dashboard, with significant improvements in styling and functionality.

# Project Module Description
- **AdminDashboard**: Central interface for managing freight data and statistics.
- **UserDashboard**: Access point for freight calculations and historical data.
- **CombinedFreightTable**: Manages combined freight rates with improved styling.
- **SeaFreightTable**: Manages sea freight rates with updated terminology.
- **AgentSeaFreightTable**: Handles agent-specific sea freight rates, incorporating carrier considerations.
- **FreightTable**: Overview of freight routes and costs with search and pagination.
- **AddRouteForm**: Interface for adding new freight routes.
- **CostCalculator**: Tool for calculating shipping costs, now displaying L.LOCAL values as deductions.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **OtherCostsTable**: Handles additional freight transport costs.
- **LoginForm**: User authentication interface.
- **Layout**: Consistent application layout with dynamic navigation.
- **InlineEditCell**: Inline editing functionality for table data.
- **UserManagement**: Manage user roles and permissions with enhanced functionality.
- **QuotationDialog**: Generate and export quotations.
- **AuditLogTable**: Displays detailed audit logs.
- **TimeMachineDialog**: View historical audit logs.
- **ValidityPeriodInput**: Manage freight rate validity periods.
- **SupabaseIntegration**: Handles connection to Supabase.
- **SupabaseMigration**: Manages initial data migration to Supabase.
- **CheckDuplicateRailFreight**: Identifies duplicate rail freight entries.
- **ExcelExport**: Export quotations to Excel with enhanced styling.
- **DestinationTable**: Manage destination entries with improved functionality.
- **BorderDestinationTable**: Component for managing border destination freight rates.
- **DPCostTable**: Updated to include the Package icon for enhanced functionality.
- **FreightLoaders**: New module for loading various freight-related data.
- **FreightCalculations**: New module for calculating costs with tracking of expired rates and detailed breakdowns.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for static typing.
- **Shadcn-UI**: Component library for UI elements.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web projects.
- **xlsx**: Library for Excel file manipulation.
- **xlsx-js-style**: Enhanced library for styled Excel file manipulation.
- **Supabase**: Backend as a service for database management and real-time data handling.
- **Vercel**: Deployment platform for hosting applications.
