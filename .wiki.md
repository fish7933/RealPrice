# Project Summary
This project is an advanced freight cost calculator designed specifically for logistics companies. It optimizes shipping cost estimations by considering various factors such as ports of loading (POL), ports of discharge (POD), weight, and additional fees. The application enhances operational efficiency and decision-making by enabling users to compute and compare freight costs across different shipping methods. Recent updates have improved the user experience, particularly in the CostResultTable component, which now features enhanced visual indicators and additional features for cost breakdown.

# Project Module Description
The project consists of several functional modules:
- **CostCalculatorWithTabs**: Core component for freight cost calculation with a tabbed interface.
- **AdminDashboard**: Interface for managing freight-related data, including agents and ports.
- **AgentSeaFreightTable**: Displays statistics of sea freight agents.
- **ShippingLineTable**: Manages and displays shipping line information.
- **DTHCTable**: Manages Destination Terminal Handling Charge (D/O) entries.
- **CombinedFreightTable**: Manages combined freight rates.
- **BorderDestinationTable**: Manages freight rates for border destinations.
- **WeightSurchargeTable**: Manages weight surcharge rules.
- **DPCostTable**: Manages direct port cost information.
- **CostCalculatorHeader**: Header component for the cost calculator.
- **CostResultTable**: Displays optimized cost calculation results, highlighting agent codes and the lowest prices, now with improved cost breakdown features.
- **CalculationHistory**: Manages and displays historical calculation data.
- **SeaFreightDialog**: Dialog for selecting sea freight options.
- **CostInputForm**: Updated form for inputting cost-related data.
- **RailAgentTable**: Displays rail freight agents.
- **TruckAgentTable**: Displays truck freight agents.
- **LoginForm**: User authentication form.
- **TimeMachineDialog**: Enhanced dialog for selecting historical dates for freight rate changes.
- **PortBorderTable**: Manages freight rates for border crossings.
- **QuotationDialog**: New dialog for generating quotations based on calculated costs and exporting to Excel.
