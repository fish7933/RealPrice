# Project Summary
The project is an advanced freight cost calculator tailored for logistics companies, enabling precise shipping cost estimates. It incorporates various parameters such as ports of loading (POL), ports of discharge (POD), weight, and additional charges. The tool enhances decision-making and operational efficiency in logistics by allowing users to calculate and compare freight costs across multiple shipping methods. Recent updates have introduced a modernized user interface with vibrant gradients, animations, and improved visual hierarchy, significantly enhancing user experience.

# Project Module Description
The project consists of several functional modules:
- **CostCalculatorWithTabs**: Core component for calculating and displaying freight costs with a tabbed interface.
- **AdminDashboard**: Interface for managing freight-related data, including agents, ports, and costs.
- **AgentSeaFreightTable**: Displays statistics of sea freight agents.
- **DTHCTable**: Manages D/O (Destination Terminal Handling Charge) entries and their versions.
- **CombinedFreightTable**: Manages combined freight rates and their details.
- **BorderDestinationTable**: Manages freight rates for border destinations.
- **WeightSurchargeTable**: Manages rules for weight surcharges.
- **DPCostTable**: Manages direct port cost information.
