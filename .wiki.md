# Project Summary
The project is an advanced freight management system designed to optimize logistics and shipping operations across various transportation modes, including sea, rail, and truck freight. It provides businesses with essential tools for cost calculation, user account management, quotation generation, and historical data tracking, ultimately enhancing operational efficiency and transparency. Recent updates have focused on improving the quotation management system, including enhancements to the memo functionality for quotations.

# Project Module Description
The project consists of several functional modules:
- **User Management**: Handles user authentication and role assignments.
- **Freight Calculation**: Computes freight costs and maintains historical data.
- **Audit Logging**: Tracks changes to freight data for traceability.
- **Data Management**: Manages freight-related data entities, including DTHC, DPCost, and weight surcharge rules.
- **Port and Destination Tables**: Manages freight rates for rail and truck transport across various destinations.
- **Combined Freight Table**: Handles combined freight rates and operations.
- **Freight Calculator**: Provides a user interface for calculating freight estimates.
- **Cost Input Form**: Allows users to input additional costs for freight calculations, including domestic transport costs.
- **Cost Result Table**: Displays results of freight calculations and cost breakdowns.
- **User Dashboard**: Displays user statistics and recent calculations.
- **Agent Management**: Manages rail and truck agents with a focus on usability.
- **Admin Dashboard**: Provides management functionalities through a tabbed interface for agents, ports, destinations, and more.
- **Time Machine Dialog**: Allows users to select historical dates and view corresponding freight audit logs.
- **Quotation Management**: Manages quotations including history, pagination, filtering, and features for viewing, exporting, and memo management.

# Usage
1. **Install Dependencies**: Run the command to install the necessary dependencies.
2. **Build the Project**: Use the build command to compile the project.
3. **Run the Application**: Start the application using the appropriate command.
4. **Interact with the Calculator**: Utilize the UI to input POL, POD, weight, and additional costs to calculate freight estimates, including the option to select historical dates.
5. **Manage Quotations**: Access the Quotation List section to view, filter, and manage quotations, as well as export them to Excel. You can also add and edit memos associated with quotations.
